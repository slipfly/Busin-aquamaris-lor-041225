# gulp-builder-v2

**tasks:**
gulp assets - _создание папки dist и перенос туда ассетов_
gulp - _для работы и сохранения изменений_
gulp delivery - _создание скриншотов, создание зипов_

**.env**
данный файл создаете самостоятельно, в корневой директории сборщика. в нем необходимо прописать переменную FIGMA_API_TOKEN=ваш-персональный-токен-фигмы (это ваши личные данные, не убирайте .env из гитигнора)

версия ноды - v20.6.0
